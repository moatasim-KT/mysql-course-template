# Week 15: BI Integration & Automation

## 1. Learning Objectives
By the end of this lesson, students will be able to:
- Connect MySQL instances to popular BI tools (Tableau/PowerBI/Looker).
- Use Python (Pandas/SQLAlchemy) to query and analyze MySQL data.
- Automate basic reporting tasks.

## 2. Concept Overview (Theory)
- **Why it matters**: Analysts rarely just use SQL Workbench. They need to integrate SQL into broader workflows and visualization tools to share insights with stakeholders.
- **Example**: Automating a daily email summary of top-selling products using a Python script that queries MySQL.

## 3. Guided Lab (MySQL Workbench)
- Setup a read-only user account for BI tool access.
- Write a Python snippet to fetch data from MySQL into a pandas DataFrame.

## 4. Independent Practice (Challenge)
1. Create a Python script that exports query results to a CSV file.
2. Configure a connection to a mock BI environment.

## 5. Assessment & Validation
- Demonstrate your Python integration script.
