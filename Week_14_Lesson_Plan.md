# Week 14: Analytical Methodologies

## 1. Learning Objectives
By the end of this lesson, students will be able to:
- Perform Cohort Analysis using SQL.
- Implement RFM (Recency, Frequency, Monetary) analysis.
- Execute Funnel/Conversion Rate analysis.

## 2. Concept Overview (Theory)
- **Why it matters**: These methodologies transform raw transaction data into actionable business insights regarding user behavior and retention.
- **Example**: Determining the retention rate of customers based on their first purchase month.

## 3. Guided Lab (MySQL Workbench)
- Write queries to calculate monthly retention cohorts from the `orders` table.

## 4. Independent Practice (Challenge)
1. Calculate the RFM score for each customer.
2. Analyze the funnel conversion (e.g., website visit -> add to cart -> checkout) using simulated logs.

## 5. Assessment & Validation
- Run the test script `tests/test_week14.py`.
