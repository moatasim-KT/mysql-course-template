# Week 16: Capstone Project

## 1. Learning Objectives
By the end of this lesson, students will be able to:
- Apply all acquired SQL skills to a comprehensive real-world dataset.
- Design, implement, and document a database-driven analysis project.
- Communicate insights professionally.
- Provide and act upon constructive peer feedback.

## 2. Concept Overview (Theory)
- **Why it matters**: Integration of all skills—from schema design and cleaning to advanced querying and BI reporting—is the goal of the course.
- **Example**: Building an end-to-end dashboard for a mock business, from data ingestion to automated reports.

## 3. Guided Lab (Capstone Guidance)
- Defining project requirements and scope.
- Sourcing and importing complex, multi-table datasets.
- Structuring the final project repository.
- **Peer Review Task**: Use the [Peer Review Rubric](docs/Peer_Review_Rubric.md) to evaluate a peer's project proposal.

## 4. Independent Practice (Challenge)
- Complete your capstone project and present your findings.

## 5. Assessment & Validation
- Final project code review, data quality check, and presentation.
- Peer review submission using the provided rubric.
